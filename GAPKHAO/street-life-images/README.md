# Street Life Images Folder

## Purpose
This folder contains images for the 5 story cards on the Street Life page.

## Image Requirements

### File Names (Use exactly these names)
1. `fried-rice-vendor.jpg` - แม่ค้าข้าวผัดตำนานย่านเยาวราช (Food)
2. `temple-alley.jpg` - ซอยวัดกลางเมือง (Culture)
3. `street-murals.jpg` - จิตรกรรมฝาผนังซอย (Art)
4. `night-market.jpg` - ตลาดกลางคืนบรรยากาศดี (Food)
5. `bangkok-night.jpg` - Bangkok After Dark (City)

### Image Specifications
- **Format**: JPG, JPEG, or PNG
- **Recommended Size**: 
  - Cards 1-4: 600px width minimum
  - Card 5 (wide): 900px width minimum
- **Aspect Ratio**: 
  - Cards 1-4: 3:4 (portrait) or 1:1 (square)
  - Card 5: 16:9 (landscape)
- **File Size**: Keep under 500KB for fast loading
- **Quality**: High quality, well-lit photos

## How to Add Your Images

### Step 1: Prepare Your Images
1. Choose 5 photos that match the descriptions above
2. Rename them exactly as listed (case-sensitive)
3. Optimize file size if needed (use online tools like TinyPNG)

### Step 2: Add to Folder
1. Copy your 5 images to this folder: `street-life-images/`
2. Make sure file names match exactly:
   - `fried-rice-vendor.jpg`
   - `temple-alley.jpg`
   - `street-murals.jpg`
   - `night-market.jpg`
   - `bangkok-night.jpg`

### Step 3: Test
1. Run `start-server.bat`
2. Open `http://localhost:8000/street-life.html`
3. Your images should appear on the cards
4. Click cards to see the story modals

## Fallback Images
If you don't add images, placeholder images will be shown with the text "Add your image here".

## Tips for Best Results

### Photography Tips
- Use high-resolution photos
- Ensure good lighting
- Capture authentic Thai street life
- Show people, food, and culture
- Avoid blurry or dark images

### Image Editing
- Adjust brightness/contrast if needed
- Crop to appropriate aspect ratio
- Compress to reduce file size
- Maintain natural colors

### Where to Find Images
- Your own photos from Thailand
- Stock photo sites (with proper license)
- Thai tourism websites
- Local photographer collaborations

## Troubleshooting

### Images Not Showing
- Check file names match exactly (case-sensitive)
- Verify images are in correct folder
- Ensure file format is JPG/JPEG/PNG
- Clear browser cache (Ctrl+F5)
- Check file permissions

### Images Look Stretched
- Check aspect ratio matches recommendations
- Use correct dimensions for each card
- Consider cropping images to fit

### Slow Loading
- Compress images (aim for <500KB each)
- Use JPG format for photos
- Optimize with tools like TinyPNG or ImageOptim
