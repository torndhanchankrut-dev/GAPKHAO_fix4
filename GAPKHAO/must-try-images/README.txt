MUST-TRY IMAGES FOLDER
======================

Place your dish images here with the following names:

1. pad-thai.jpg - Pad Thai dish image
2. som-tam.jpg - Papaya Salad dish image
3. boat-noodles.jpg - Boat Noodles dish image
4. mango-sticky-rice.jpg - Mango Sticky Rice dish image

Recommended image specifications:
- Format: JPG or PNG
- Size: 800x600px or higher
- Aspect ratio: 4:3 or similar
- Quality: High resolution for best display

The design will automatically use these images when placed in this folder.
